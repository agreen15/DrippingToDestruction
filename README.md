# DrippingToDestruction
This is the code repository for the model described in the manuscript by Green &amp; Cooper (2022) under consideration for publication at EPSL.

To use this model, download the .ipynb file meant for use in the Jupyter Notebooks scripting tool associated with Underworld 2 (Mansour et al., 2020).
For the results included in Green & Cooper (2022), please modify the values and flags in the section "Flags, Switches, and Variables: Change These!".
